This folder contains the files for my SWLightSaberStore dataset. 
The data compiled for this project is not available online as it was created by me. 

The purpose of the project: 
1. Choose a fun dataset.
2. Create relational tables from it. 
3. Insert data into those tables via DML statements.
4. Create queries and views.
5. Create an interactive Power BI Dashboard from queries and views.

Questions answered from the data.

Who are our customers?

How many orders were placed per customer?

How many of those customers are new?

What are the total sales and total orders by month?

What are the most popular products and the product color combinations?
